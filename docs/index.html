CoCreate-select

Used to build fascinating select html elements

Install

 
npm i @cocreate/select

Or you can use cdn link:

<script>https://cdn.cocreate.app/select/latest/CoCreate-select.min.js</script>

usage

This is select usage

                                
------------ Single Options ------------
<cocreate-select class="sortable floating-label" 
  name="selectField4" 
  placeholder="Single Options">
  <input class="option-search" placeholder="Search">
  <ul class="options  overflow:overlay">
		<li class="option"  data-value="value 1">
		  <h3 >value 1</h3>
		</li>
		<li class="option"  data-value="value 2">
		  <h3 >value 2</h3>
		</li>
		<li class="option"  data-value="value 3">
		  <h3 >sss</h3>
		</li>
		<li class="option"  data-value="value 3">
		  <h3 >value 3</h3>
		</li>
  </ul>
</cocreate-select>

------------ Fetch array _id ------------

<cocreate-select class="sortable floating-label" 
  name="selectField6" collection="test" 
  document_id="5e8ff00e8c3af32574df80cb"  
  placeholder="Fetch array _id" 
  data-limit_option="3" multiple>
  <input class="option-search" placeholder="Search" template_id="tempplate6">
  <ul fetch-collection="test" 
    fetch-document_id="5e8ff00e8c3af32574df80cb" 
    fetch-name="selectField1" 
    id="array02"
    pass_to="me1234" 
    class="template-wrapper-array options overflow:overlay">
    <li class="template option" pass_id="me1234">
      <h3 collection="users" document_id="" name="name" pass_id="me1234" >Organization Name</h3>
      <h5 collection="users" document_id="" name="_id" pass_id="me1234" >id</h5>
    </li>
  </ul>
</cocreate-select>

This is select usage

Attributes ul into select

  • fetch-collection string required

    collection in db

  • fetch-document_id string required

    id in db

  • fetch-name string required

    attribute in collection in ours db

  • pass_to string required

    Template to render response
