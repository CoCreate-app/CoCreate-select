CoCreate-select

Used to build fascinating select html elements

Install

 
npm i @cocreate/select

Or you can use cdn link: 

<script>https://cdn.cocreate.app/select/latest/CoCreate-select.min.js</script>

usage

This is select usage 

                                
------------ Single Options ------------
<cocreate-select class="sortable floating-label" 
  key="selectField4" 
  placeholder="Single Options">
  <input class="option-search" placeholder="Search">
  <ul class="options  overflow:auto">
		<li class="option"  value="value 1">
		  <h3 >value 1</h3>
		</li>
		<li class="option"  value="value 2">
		  <h3 >value 2</h3>
		</li>
		<li class="option"  value="value 3">
		  <h3 >sss</h3>
		</li>
		<li class="option"  value="value 3">
		  <h3 >value 3</h3>
		</li>
  </ul>
</cocreate-select>

------------ Fetch array _id ------------

<cocreate-select class="sortable floating-label" 
  key="selectField6" array="test" 
  object="5e8ff00e8c3af32574df80cb"  
  placeholder="Fetch array _id" 
  data-limit_option="3" multiple>
  <input class="option-search" placeholder="Search" template_id="tempplate6">
  <ul array="test" 
    object="5e8ff00e8c3af32574df80cb" 
    key="selectField1" 
    id="array02"
    pass_to="me1234" 
    class="options overflow:auto">
    <li class="option" pass_id="me1234">
      <h3 array="users" object="" key="name" pass_id="me1234" >Organization Name</h3>
      <h5 array="users" object="" key="_id" pass_id="me1234" >id</h5>
    </li>
  </ul>
</cocreate-select>

Attributes ul into select

  • array string required

    array in db

  • object string required

    id in db

  • name string required

    attribute in array in ours db

  • pass_to string required

    Template to render response
